# ClipFlow 项目使用

## 环境要求

- Node 版本建议：`22.x LTS`
- 推荐版本：`22.16.0`
- 包管理器：`npm`

## 安装依赖

```bash
npm install
```

## 启动开发环境

```bash
npm run dev
```

启动后将完成：

- Electron 主进程构建
- preload 构建
- renderer 开发服务启动
- Electron 桌面窗口启动

## 构建项目

```bash
npm run build
```

构建成功后会生成：

- `out/main`
- `out/preload`
- `out/renderer`

## 调试方式

开发环境下已支持调试模式：

- 启动后自动打开 DevTools
- 支持 `F12`
- 支持 `Ctrl + Shift + I`

## 当前目录说明

```text
ClipFlow/
  main/       Electron 主进程
  preload/    预加载桥接
  renderer/   Vue3 渲染层
  shared/     跨层共享类型与常量
  docs/       项目文档沉淀
```

## 页面入口说明

- HTML 入口：`renderer/index.html`
- Vue 启动入口：`renderer/src/main.ts`
- 首页根组件：`renderer/src/App.vue`

## 常见排查入口

### 页面白屏

优先检查：

- preload 是否正常加载
- DevTools Console 是否有红色报错
- `window.clipflow` 是否可用

### 启动失败

优先检查：

- Node 版本是否满足要求
- `electron.vite.config.ts` 入口配置是否完整
- 依赖是否安装完成

### 构建失败

优先检查：

- TypeScript 类型错误
- 路径别名配置
- 入口文件路径是否变更
