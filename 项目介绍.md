# ClipFlow 项目介绍

## 项目背景

`ClipFlow` 是一个用于求职补强和个人能力沉淀的桌面项目。目标不是做成复杂工具平台，而是通过一个可运行、可打包、可演示的桌面工具，完整串起 Electron 主进程、预加载桥接、渲染层交互和基础工程化能力。

## 一句话定义

`ClipFlow` 是一个基于 `Electron + Vue3 + TypeScript` 开发的桌面效率工具，用于管理剪贴板历史、执行常见文本转换，并维护可复用的常用片段。

## 项目定位

- 面向个人单机使用
- 聚焦高频文本处理场景
- 优先保证功能闭环和可复述性

## 核心功能范围

### 1. 剪贴板历史

- 自动记录文本剪贴板内容
- 连续重复内容不重复入库
- 展示复制时间
- 一键复制回系统剪贴板
- 删除单条记录
- 清空普通记录
- 置顶 / 取消置顶

### 2. 文本转换

- `trim`
- 多行转单行
- 单行按分隔或换行拆分
- 大写转换
- 小写转换
- JSON 格式化
- JSON 压缩
- 转换结果一键复制
- 非法 JSON 明确提示错误

### 3. 常用片段管理

- 定位为个人本地常用文本片段库，不是剪贴板收藏夹或笔记系统
- 用于保存会反复复制使用的固定文本，如日报模板、联调说明、常用命令片段
- 新增片段
- 编辑片段
- 删除片段
- 搜索片段
- 一键复制片段内容
- 使用“标题 + 内容”结构
- 本地持久化保存

## 技术选型

- 桌面容器：`Electron`
- 前端界面：`Vue3`
- 语言：`TypeScript`
- 构建工具：`Vite`
- 工程组合：`electron-vite`
- 状态管理：首版优先轻量方案，必要时再引入 `Pinia`
- 持久化方案：本地文件或轻量方案，优先简单稳定

## 当前不做什么

- 账号登录
- 云同步
- AI 功能
- 图片剪贴板
- 文件剪贴板
- 富文本剪贴板
- 复杂标签体系
- 收藏体系
- 笔记系统或知识库能力
- 插件系统
- 快捷键配置中心
- 多窗口模式
